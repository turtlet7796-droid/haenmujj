** 한의원 홈페이지 맞춤 제작용 프롬프트

당신은 10년 차 전문 웹 퍼블리셔이자 UI/UX 디자이너입니다.
아래 제공하는 우리 한의원의 정보를 바탕으로, 방문자가 신뢰감을 느낄 수 있는 세련된 반응형 홈페이지 코드(HTML과 CSS)와 비급여목록표 코드를 작성해 주세요.
코드는 PC와 모바일 화면 모두에서 완벽하게 보이도록 구성해야 합니다.

[1. 한의원 기본 정보]

1. 한의원 이름: [예: 해나무한의원]

2. 한의원 주소 : [예: 전주시 덕진구 솔내로 152, 푸른시티 2층]

3. 핵심 캐치프레이즈: [예: 척추 관절, 수술 없이 바르게 치료합니다]

4. 주요 진료 과목: [예: 1. 교통사고 후유증, 2. 추나요법, 3. 첩건한약, 4. 산재치료]

5. 진료 시간 특징: [예: 월,수,금 야간진료(오후 8시 30분까지), 일요일진료(오후 1시~5시)]

6. 사업자 정보 (푸터용): [예: 대표자 한상희 / 사업자등록번호 232-92-00335 / 대표전화 063-277-6688 ]


[2. 원하는 디자인 및 스타일]

1. 메인 컬러 (포인트 색상): [예: 신뢰감을 주는 네이비 블루 / 따뜻하고 활기찬 오렌지 / 편안한 숲속의 초록색]

2. 전체적인 분위기: [예: 모던하고 깔끔한 느낌 / 전통적이고 따뜻한 느낌 / 호텔처럼 고급스러운 느낌]

3. 폰트 스타일: 가독성이 좋은 고딕계열 폰트(Pretendard 등) 사용

[3. 홈페이지에 꼭 들어갔으면 하는 구성 (섹션)]

1. 메인 대문 (Hero): 한의원 이름과 핵심 문구, 배경 이미지가 들어갈 자리

2. 진료 철학: 환자를 대하는 원장님의 진심을 담은 짧은 글

3. 의료진 소개: 원장님 사진이 들어갈 공간과 주요 약력 (전문의 여부 등)

4. 진료 과목 안내: 주요 진료 과목들을 한눈에 보기 쉽게 그리드(표) 형태로 배치

5. 진료 시간표: 월~일요일까지의 진료 시간을 알기 쉬운 표 형태로 제공

6. 오시는 길 (지도): 네이버 지도 링크와 상세 주소, 대중교통/주차 안내

[4. 필수 추가 기능]

1. 플로팅 버튼: 모바일과 PC 화면 우측 하단에 화면을 따라다니는 [네이버 예약하기] 버튼과 [전화 문의하기] 버튼을 만들어 주세요.

2. 모바일 최적화: 모바일 화면(너비 768px 이하)으로 접속 시, 글씨가 너무 작아지지 않고 터치하기 편하게 레이아웃이 1열로 자동 변환되도록 CSS 미디어 쿼리를 세밀하게 작성해 주세요.

3. 푸터 (하단 정보란): 홈페이지 맨 아래에 한의원 주소, 사업자등록번호, 대표자명, 연락처, 이메일 등을 깔끔하게 정돈한 푸터(Footer) 영역을 만들어 주세요. 한쪽에는 '비급여목록표'로 넘어가는 링크도 달아주세요.
